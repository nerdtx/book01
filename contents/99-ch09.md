## Section 9: Assignment – Create a Simple Contact List

In this assignment, you’ll build a simple contact list using **arrays of
objects** and **loops**. This is a hands-on way to practice all the data
structure skills you’ve learned in this chapter.

### Assignment Overview

You will write a JavaScript program that:

* Stores contact data using an **array of objects**
* Uses a **loop** to display each contact’s name and email
* Allows a user to add a new contact with `prompt()`
* Outputs updated contact data using `console.log()`

###  Starting Data

Start with an array like this:

```js
let contacts = [
  { name: "Taylor", email: "taylor@example.com" },
  { name: "Riley", email: "riley@example.com" }
];
```

###  Requirements

Your program must:

1. **Loop through the contacts** and print each one in a clear format.
2. **Ask the user for a new contact** using `prompt()` (name and email).
3. **Add the new contact** to the list.
4. Print the entire updated list of contacts.

You can structure your output like this:

```
Contact List:
Name: Taylor, Email: taylor@example.com
Name: Riley, Email: riley@example.com
Name: Alex, Email: alex@example.com
```

###  Bonus Challenge (Optional)

After adding the contact, allow the user to search for a contact by name:

```js
let searchName = prompt("Enter a name to search:");
```

Then loop through the list and print the contact if it exists.

### Example Output

```
Contact List:
Name: Taylor, Email: taylor@example.com
Name: Riley, Email: riley@example.com

Enter a new name: Alex
Enter a new email: alex@example.com

Updated Contact List:
Name: Taylor, Email: taylor@example.com
Name: Riley, Email: riley@example.com
Name: Alex, Email: alex@example.com
```
