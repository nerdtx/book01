## Section 2: Selecting Elements from the Page

Now that you understand how the browser sees your page as a tree of elements in
the DOM, the next step is learning how to **access those elements** with
JavaScript.

### The `document` Object

The browser gives JavaScript a built-in object called `document`. This object
represents the entire HTML document. From it, you can select anything on the
page.

For example:

```js
console.log(document);
```

This will print the entire DOM structure to the browser console.

### Selecting by ID

If an element in your HTML has an `id`, you can use `getElementById` to select
it:

```html
<p id="welcome">Hello!</p>
```

```js
let message = document.getElementById("welcome");
console.log(message);
```

### Selecting by Class or Tag Name

If you want to select multiple elements, use:

* `getElementsByClassName("some-class")`
* `getElementsByTagName("p")`

These methods return **HTMLCollections** - a type of list of matching elements.

### Using `querySelector`

There also exists a more modern, flexible way to select elements:

```js
document.querySelector("css-selector");
```

Examples:

```js
let header = document.querySelector("h1");
let box = document.querySelector(".box");      // class
let login = document.querySelector("#login");  // id
```

To get multiple elements, use:

```js
document.querySelectorAll("css-selector");
```

> **Helpful Hint:**
> `querySelector` uses the same syntax as CSS selectors. That means your
> experience writing CSS directly helps you grab DOM elements too.

### Practice Example

Given this HTML:

```html
<h1 id="main-title">Welcome!</h1>
<p class="tagline">Your adventure begins here.</p>
```

Try these JavaScript statements in the browser console:

```js
let title = document.querySelector("#main-title");
let tag = document.querySelector(".tagline");

console.log(title.textContent);  // "Welcome!"
console.log(tag.textContent);    // "Your adventure begins here."
```

elements after selecting them.
