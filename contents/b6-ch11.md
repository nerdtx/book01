## Section 6: Interactive Web Example – A Greeting Form

Let’s put together the skills you’ve learned so far in this chapter by building
a small interactive webpage. This project will walk you through creating a
simple greeting form that:

* Collects the user’s name and favorite animal
* Responds with a custom message when the form is submitted
* Optionally gives feedback while typing (stretch goal)

This is your first complete interactive JavaScript experience tied to the page’s
structure.

### Step 1: Create the HTML Structure

Here’s a basic HTML form that asks for two pieces of information.

```html
<h2>Greeting Generator</h2>

<form id="greetingForm">
  <label for="nameInput">Your Name:</label>
  <input id="nameInput" type="text" placeholder="e.g. Maria"><br><br>

  <label for="animalInput">Favorite Animal:</label>
  <input id="animalInput" type="text" placeholder="e.g. Llama"><br><br>

  <button type="submit">Greet Me</button>
</form>

<p id="greetingMessage"></p>
```

### Step 2: Add the JavaScript

Now let’s add the script that listens for form submission and displays a
message.

```html
<script>
  const form = document.getElementById("greetingForm");
  const nameInput = document.getElementById("nameInput");
  const animalInput = document.getElementById("animalInput");
  const message = document.getElementById("greetingMessage");

  form.addEventListener("submit", function(event) {
    event.preventDefault();  // Prevents page reload

    const name = nameInput.value;
    const animal = animalInput.value;

    message.innerText = `Hi ${name}! It's awesome that you like ${animal}s.`;
  });
</script>
```

### Step 3: Style It (Optional)

If you’d like to add a little polish, try these CSS rules:

```html
<style>

    font-family: sans-serif;

  }

    width: 100%;
    margin-bottom: 0.5rem;
  }

    margin-top: 1rem;
    font-weight: bold;
  }
</style>
```

> **Helpful Hint:**
> If the message doesn’t appear, check that your element IDs match between the
> HTML and JavaScript.

### Stretch Feature: Live Preview

Want to see the message update as the user types? Try listening for `"input"`
events:

```js
nameInput.addEventListener("input", updateMessage);
animalInput.addEventListener("input", updateMessage);

function updateMessage() {
  const name = nameInput.value;
  const animal = animalInput.value;

  if (name && animal) {
    message.innerText = `Hi ${name}! It's awesome that you like ${animal}s.`;
  } else {
    message.innerText = "";
  }
}
```

This builds a live, reactive experience - one of JavaScript’s most exciting
strengths.
