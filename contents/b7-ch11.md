## Section 7: Assignment – Interactive Mad Libs

It’s time to use what you’ve learned in this chapter to create a fun, silly,
interactive web application! In this assignment, you’ll build your own **Mad
Libs generator** using HTML and JavaScript. This will give you hands-on
experience working with:

* HTML forms
* Event listeners
* DOM manipulation
* Dynamic text replacement

### Assignment Goal

Create a small webpage that asks the user to input a few words (like a noun,
verb, and adjective), and then uses JavaScript to generate a goofy story using
those words.

### Requirements

#### 1. **HTML Form**

Your form should include:

* One input for a **noun** (e.g., "cat")
* One input for a **verb** (e.g., "jump")
* One input for an **adjective** (e.g., "sparkly")
* A submit button

Make sure each input has an appropriate `id` and label.

#### 2. **JavaScript Logic**

* When the user clicks the submit button:

  * Prevent the page from refreshing.
  * Read the values from the form fields.
  * Replace placeholder words in a story with the words the user entered.
  * Display the story in a `<p>` or `<div>` on the page.

#### 3. **Output Area**

* Add a place on the page for the finished story to be shown.
* Make it visually distinct using CSS (e.g., a border or background color).

#### 4. **Example Template Story**

You can use this simple story template to start:

> "Once upon a time, a very **[adjective]** **[noun]** decided to **[verb]** all
> the way to school."

When the user submits the form with `adjective = silly`, `noun = dog`, and
`verb = dance`, the output would be:

> "Once upon a time, a very **silly** **dog** decided to **dance** all the way
> to school."

> **Helpful Hint:**
> Wrap your replacement text inside a `span` tag with a class to make the story
> fun to style with CSS.

### Stretch Goals (Optional)

* Add more input fields (e.g., another noun or an adverb).
* Let the user pick between multiple story templates.
* Add images or emojis that match the input values.
* Allow the user to reset and try again.
