## Section 6: Build a Basic Notes App (Hands-On)

In this section, you will build your first complete single page application.

This Notes App will let users:

* Add new notes
* View existing notes
* Delete notes
* Switch between screens
* Save notes using `localStorage`

This project combines everything you’ve learned so far into one practical
application.

### Step 1: Plan the App

Before writing code, decide how the app will work.

Your app will have:

* A **Home View**
* A **Notes View**
* A **New Note View**

Your data will be stored in an array:

```js
let notes = [];
```

Each note will be a simple string.

### Step 2: Create the HTML Structure

Start with this basic layout:

```html
<nav>
  <button id="homeBtn">Home</button>
  <button id="notesBtn">Notes</button>
  <button id="newBtn">New Note</button>
</nav>

<div id="homeView" class="view">
  <h2>Welcome</h2>
  <p>Your personal notes app.</p>
</div>

<div id="notesView" class="view hidden">
  <h2>Your Notes</h2>
  <ul id="notesList"></ul>
</div>

<div id="newView" class="view hidden">
  <h2>Add Note</h2>
  <input id="noteInput" type="text">
  <button id="saveBtn">Save</button>
</div>
```

Add this CSS:

```css
.hidden {
  display: none;
}

.view {
  padding: 1rem;
}
```

### Step 3: Set Up Navigation

Add JavaScript to switch between views:

```js
function showView(id) {
  document.querySelectorAll(".view").forEach(v => {
    v.classList.add("hidden");
  });

  document.getElementById(id).classList.remove("hidden");
}

homeBtn.onclick = () => showView("homeView");
notesBtn.onclick = () => showView("notesView");
newBtn.onclick = () => showView("newView");
```

### Step 4: Add Notes

Create functions to manage notes:

```js
let notes = [];

function addNote(text) {
  notes.push(text);
  saveNotes();
  renderNotes();
}

function renderNotes() {
  let list = document.getElementById("notesList");
  list.innerHTML = "";

  notes.forEach((note, index) => {
    let li = document.createElement("li");
    li.textContent = note;

    let del = document.createElement("button");
    del.textContent = "Delete";
    del.onclick = () => deleteNote(index);

    li.appendChild(del);
    list.appendChild(li);
  });
}
```

### Step 5: Save and Load Data

Use `localStorage` to store notes:

```js
function saveNotes() {
  localStorage.setItem("notes", JSON.stringify(notes));
}

function loadNotes() {
  let data = localStorage.getItem("notes");

  if (data !== null) {
    notes = JSON.parse(data);
  }
}
```

Call `loadNotes()` when the page loads.

### Step 6: Connect the Form

Handle saving new notes:

```js
saveBtn.onclick = () => {
  let text = noteInput.value.trim();

  if (text === "") {
    alert("Enter a note first.");
    return;
  }

  addNote(text);
  noteInput.value = "";
  showView("notesView");
};
```

> **Helpful Hint:**
> Always trim user input before saving. This prevents empty or space-only notes.

### Step 7: Test Your App

Try:

* Adding notes
* Reloading the page
* Deleting notes
* Switching views

If everything works, you’ve built a real single page app.
