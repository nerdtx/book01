## Section 4: Responding to Events

Modern websites don’t just sit there — they **respond** to users. JavaScript
makes this possible through **event handling**. An *event* is something that
happens — a user clicks a button, presses a key, moves the mouse, submits a
form, and so on.

### What Is an Event Listener?

An **event listener** is a function that waits for a specific event to occur and then executes some code in response.

Use `.addEventListener()` to attach a function to an element:

```js
let button = document.querySelector("#submit-btn");

button.addEventListener("click", function () {
  alert("Button clicked!");
});
```

This listens for a `"click"` on the `#submit-btn` and runs the function when it happens.

### Common Events

| Event Name    | Description                           |
| ------------- | ------------------------------------- |
| `"click"`     | User clicks an element                |
| `"mouseover"` | Mouse hovers over an element          |
| `"mouseout"`  | Mouse leaves an element               |
| `"keydown"`   | User presses a key                    |
| `"submit"`    | Form is submitted                     |
| `"change"`    | Value of input changes (e.g., select) |

### Anonymous vs Named Functions

You can write the event handler inline, or define it separately:

```js
function handleClick() {
  alert("Clicked!");
}

button.addEventListener("click", handleClick);
```

Both approaches are valid — choose whichever makes the code easier to read.

> **Helpful Hint:**
> If your event function is long or reused, give it a name. If it's short and only used once, inline is fine.

### Removing Event Listeners

Sometimes, you may want to remove a listener:

```js
button.removeEventListener("click", handleClick);
```

This only works with named functions — not with anonymous functions.

### Practice Example

```html
<button id="change-color">Change Background</button>
```

```js
let button = document.getElementById("change-color");

button.addEventListener("click", function () {
  document.body.style.backgroundColor = "lightyellow";
});
```

Each click changes the background color of the whole page.
