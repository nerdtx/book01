## Section 3: Changing Content and Style

Now that you know how to select elements from the DOM, it’s time to learn how to
**modify** them. This includes changing the content users see and updating how
elements look using JavaScript.

### Changing Text Content

To change the text inside an element, use the `.textContent` or `.innerText`
property.

**Example:**

```html
<p id="greeting">Hello!</p>
```

```js
let greeting = document.getElementById("greeting");
greeting.textContent = "Welcome to our site!";
```

This replaces "Hello!" with "Welcome to our site!"

### Changing HTML Content

You can change the HTML inside an element using `.innerHTML`:

```js
greeting.innerHTML = "<strong>Welcome!</strong>";
```

Be careful: using `.innerHTML` allows HTML injection, so it's best used
sparingly or when you control the content.  HTML injection is where an attacker
injects some sort of malicious content into your webpage which would tehn be
viewed with the same permissions as your user.


### Changing Style with `.style`

You can change any CSS property using the `.style` object on a DOM element:

```js
greeting.style.color = "blue";
greeting.style.fontSize = "24px";
```

Property names use **camelCase** instead of dashes:

* `font-size` becomes `fontSize`
* `background-color` becomes `backgroundColor`

> **Helpful Hint:**
> JavaScript doesn’t use hyphens in property names because the minus sign (`-`)
> would be confused for subtraction.

### Hiding and Showing Elements

You can control whether an element is visible with the `.style.display`
property:

```js
greeting.style.display = "none";     // hides the element
greeting.style.display = "block";    // shows the element
```

Other common values include `"inline"` and `"flex"` depending on layout needs.

### Practice Example

Start with this HTML:

```html
<h2 id="main-title">Original Title</h2>
<button id="change-btn">Click Me</button>
```

Then use this JavaScript:

```js
let title = document.querySelector("#main-title");
let button = document.querySelector("#change-btn");

button.addEventListener("click", function () {
  title.textContent = "Title Changed!";
  title.style.color = "green";
});
```

Now, when the button is clicked, the heading changes text and color.
